# Deep Literature Standard

Use this standard before drafting Introduction or Discussion for a standard research article. The goal is to prevent shallow literature use and to make deep research answer four questions:

1. What is already known?
2. What is still specifically unanswered?
3. Which methods and claim boundaries are accepted by the field?
4. Where does the current dataset make a bounded contribution?

## Minimum Source Depth

For a standard empirical research article, target at least 30 verified sources before drafting the full manuscript.

Use fewer only when the chosen deliverable is a short communication, data note, monitoring report, or local management report, and state that downgrade clearly.

Suggested distribution:

| Source function | Minimum count | Purpose |
|---|---:|---|
| Global or synthesis background | 3-5 | Protected areas, biodiversity monitoring, human disturbance, edge effects, community response |
| Nearest-neighbor studies | 8-12 | Same taxon, method, landscape type, region, protected-area edge, or disturbance gradient |
| Method and inference-boundary papers | 5-8 | Camera traps, RAI limits, occupancy/detectability, HMSC, centroid/ordination, diversity metrics |
| Regional and species-context papers | 6-10 | Local landscape, focal protected area, regional fauna, habitat connectivity, land use, conservation status |
| Management and policy sources | 3-5 | Protected-area targets, threatened species lists, OECM/protected-area reporting, local policy context |
| Software/data citations | as needed | Analytical packages, databases, official reports |

## Literature Matrix Requirements

The literature matrix must have more than bibliographic entries. Each source needs a job.

```md
| Source ID | Citation | Source type | Function | Key finding / rule | What it already answers | Remaining gap it leaves | Manuscript location | Verified? |
|---|---|---|---|---|---|---|---|---|
| L001 |  | synthesis / nearest-neighbor / method / regional / species / policy / software |  |  |  |  | Introduction / Methods / Discussion / References only | yes / no |
```

## Search Coverage Checklist

Search across these axes until the literature matrix can support a coherent argument:

- Global protected-area effectiveness and protected-area edge literature.
- Human pressure, access, roads, community forest use, livestock, dogs, forest product collection, and understory management.
- Camera-trap inference limits: RAI, detection probability, camera placement, body size/activity bias, density vs record rate.
- Community-level methods: richness, Shannon diversity, occupancy, HMSC, ordination, centroid analysis, uncertainty.
- Regional literature: the study area, neighboring protected areas, biogeographic region, focal threatened species, land-use history.
- Recent articles from the last 5-8 years plus classic method papers.

## Gap Extraction

Do not write "few studies exist" as the gap. Convert the literature into a specific gap:

Weak:

- Few studies have examined mammals in this area.

Stronger:

- Existing regional work has addressed habitat connectivity, focal species habitat, or land-use context, but has not yet described how medium- and large-bodied mammal camera records vary along a protected-area edge gradient and local human-record index within this transition landscape.

## Passing Standard

Deep literature research passes only if it produces:

- At least 30 verified sources for a standard research article, or a justified lower-source deliverable.
- A source-function matrix.
- A "published answer vs remaining gap" summary.
- A method-boundary summary.
- A discussion-comparison map linking each major result to 2-4 comparison sources.
- A list of citations that must appear in Introduction, Methods, Discussion, and References.

## Downgrade Rule

If fewer than 20 verified relevant sources can be found for a research-article framing, or if the closest literature already answers the proposed question, downgrade to a monitoring baseline, data note, short communication, or local conservation report.
