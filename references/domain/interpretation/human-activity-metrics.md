# Human Activity Metrics — Interpretation Rules

## Definition Requirements

When discussing human RAI or human activity indices:
- Define what event types are included.
- Define what event types are excluded or separated (staff, patrol, livestock, dogs, vehicles, unknown people, community residents).
- State the zero structure and sample-size limitation when many sites have zero records.

## Non-Significance Language

Interpret non-significance as:
> "no clear statistical association detected under the current sample structure and metric definition"

**Not** as "no effect."
